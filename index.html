Estrutura básica do HTML

Já com o Visual Studio Code instalado, o primeiro passo é criar um arquivo que por convenção se chama index.html, que será a sua página inicial.

O HTML consiste em basicamente 3 partes:

  1. <html></html>
  2. <head></head>
  3. <body></body>

No html será onde vai ter todo o seu código, que será onde o navegador vai entender que é de fato um HTML

No head será o cabeçalho da sua página e no bodyFalando sobre as tags será todo o texto que o leitor irá visualizar

Sua estrutura é basicamente da seguinte forma:

<html>

........<head>

........</head>

........<body>

........</body>

</html>

Falando sobre as tags?

As tags são muito importantes para que você possa criar na sua página títulos, sub-títulos, parágrafos, citações, listas, marcações etc

As tags sempre terão o formado de <tag> e aqui no meio será o seu conteúdo </tag>

Começando pela tag de h1 até h6, elas representam o título da sua página, tendo em consideração que h1 é o maior e h6 é o menor

Por exemplo, o título dessa página é Módulo I - Primeiros passos com HTML e para que o título ficasse grande no início da página, foi utilizado a tag h1, ficando da seguinte forma <h1>Módulo 1 - Primeiros Passos com HTML</h1> e como esse texto é algo que o leitor precisa visualizar, então a tag ficou dentro da tag body

A próxima tag é a <p></p>, que representa um parágrafo. Então se você quer escrever apenas um texto normal, precisará apenas escrever o texto dentro da tag <p>aqui vai o seu texto</p>

Você pode deixar um texto em negrito utilizando a tag <strong>aqui vai o seu texto</strong>

Você pode deixar um texto pequeno utilizando a tag <small>aqui vai o seu texto</small>

Você pode deixar um link https://code.visualstudio.com/ utilizando a tag <a href="url" target="_blank">aqui vai o texto do link</a>

Pode ser que você queira marcar algo importante da sua página para chamar atenção so seu leitor.

Você pode simplesmente marcar como um marca-texto utilizando <mark>aqui vai o seu texto</mark>

Você pode simplesmente sublinhar com um underline utilizando <u>aqui vai o seu texto</u>

Você pode também mudar a cor do texto utilizando <font color="colorName">aqui vai o seu texto</font> e em colorName você digita a cor em inglês, no meu caso eu digitei "red"

Você também pode até mesmo criar um delimitador como o abaixo na cor cinza utilizando a tag <hr>

Entendendo a diferença entre listas ordenadas e não ordenadas

A lista ordenada é uma lista que vai seguir uma sequencia numérica em colunas como no exemplo abaixo

  1. Item
  2. Item

E a lista não ordenada já não tem o numerador como no exemplo abaixo

  • Item
  • Item

Para criar a lista ordenada basta utilizar

<ol>

........<li>Item</li>

........<li>Item</li>

</ol>
